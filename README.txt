## Paquetes necesarios:
    - numpy (pip install numpy)
    - matplotlib (pip install matplotlib)
    - pandas (pip install pandas)
    - sklearn (pip install scikit-learn)
    - tqdm (pip install tqdm)

## Instrucciones de ejecucion:
    - En la celda de carga de datos, la segunda del código, cambiar la ruta donde se encuentra la base de datos "emnist_letters_tp.csv"
    a aquella de su computadora para que el código pueda leerla
    - En la sección 1, ejecutar las celdas observando los gráficos generados.
    - Para poder ejecutar la sección 2 o 3, es necesario haber ejecutado la sección 1, pues renombra los atributos de la base de datos.
    - En la sección 2, ejecutar las celdas observando gráficos y print's.
    - En la sección 3, ejecutar las celdas observando print's.
